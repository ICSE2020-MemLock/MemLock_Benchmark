NAME

SWFMovie_startSound - Starts playing event-sound (SWFSound)

	SWFMovie movie,
	SWFSound sound
);

PARAMETERS

SWFMovie movie
Not Documented.
SWFSound sound
Not Documented.

DESCRIPTION
